## 模组简介

`Pangolin Survival` 是一个基于 Fabric 的生存玩法模组，核心思路是把普通挖矿和地下行动替换成“穿山甲式”的潜地与掘进机制。

当模组启用后，玩家不再使用传统镐子进行正常挖矿，而是依靠：

- 幽潜模式：在可潜入的地下介质中穿行
- 掘进模式：通过头盔等级和体力系统自动掘进
- 耐力系统：限制潜地、掘进、放置方块与脱困
- 头盔分级：不同头盔决定可挖方块等级与耐力消耗

这个模组更适合做：

- 生存 RPG 玩法
- 地下探索主题玩法
- 限制常规挖矿、强化地下风险的整合包

## 运行环境与对应版本

本模组当前工程对应的目标环境如下：

- Minecraft：`1.21.11`

## 安装建议

建议将模组同时安装在：

- 服务端
- 客户端

原因：

- 服务端负责玩法逻辑、状态判断、潜地/掘进规则
- 客户端负责按键、HUD 显示、耐力升级界面、视觉状态同步

如果只装服务端，不装客户端：

- 玩家将没有本模组按键
- HUD 不会显示
- 升级界面无法正常打开
- 整体体验不完整

## 快速开始

### 1. 安装模组

将模组 Jar 放入：

- 客户端 `mods` 文件夹
- 服务端 `mods` 文件夹

### 2. 启动游戏/服务器

首次启动后，模组会生成配置文件：

- 配置文件位置：`config/pangolin-survival.json`

### 3. 启用模组规则

本模组默认 `globalEnabled` 为 `false`，也就是默认不开启玩法。

你可以用以下两种方式启用：

#### 方式 A：命令启用

```mcfunction
/pangolin global on
```

#### 方式 B：配置文件启用

将配置中的：

```json
"globalEnabled": true
```

保存后执行：

```mcfunction
/pangolin reload
```

## 按键说明

默认按键如下：

- `G`：切换幽潜模式
- `V`：切换掘进模式
- `H`：打开耐力升级界面

你也可以在 Minecraft 的按键设置里重新绑定。

## 指令说明

### 查看状态

```mcfunction
/pangolin status
```

用途：

- 查看当前模式
- 查看当前耐力
- 查看当前头盔等级
- 管理员还能看到全局启用状态

### 启用全局规则

```mcfunction
/pangolin global on
```

用途：

- 启用整套穿山甲玩法规则

### 关闭全局规则

```mcfunction
/pangolin global off
```

用途：

- 关闭整套穿山甲玩法规则
- 玩家会被强制结束潜地状态

### 重载配置

```mcfunction
/pangolin reload
```

用途：

- 重载 `pangolin-survival.json`

说明：

- `global on/off` 与 `reload` 需要管理员权限
- `status` 普通玩家也可以使用

## 玩法机制说明

### 1. 新手帽

当模组已启用时：

- 玩家首次进入世界时会获得一顶新手帽
- 玩家死亡后若身上没有新手帽，会再次补发
- 玩家活着时如果把新手帽丢掉，不会自动补发

新手帽本质上是：

- 一顶带特殊标记的皮革帽
- 不可破坏

### 2. 幽潜模式

按 `G` 进入或退出幽潜模式。

幽潜模式特点：

- 允许玩家在可潜入介质中穿行
- 更偏向“潜地移动”，不负责自动挖掘
- 会进入地下穿行状态并允许穿过普通可潜入方块

进入幽潜模式需要满足：

- 全局规则已启用
- 玩家不在载具上
- 玩家不在水里
- 玩家不在岩浆里
- 玩家附近存在可潜入介质

### 3. 掘进模式

按 `V` 进入或退出掘进模式。

掘进模式特点：

- 玩家会朝前方目标进行自动掘进
- 破坏速度与可挖等级由当前头盔决定
- 破坏方块会消耗体力

进入掘进模式额外需要满足：

- 头盔具备掘进能力
- 当前体力足够支付进入成本

如果头盔不够或体力不足：

- 无法进入掘进模式
- 或在掘进过程中退回到幽潜模式

### 4. 出土机制

离开地下时会进入“出土中”状态。

当前版本的出土逻辑：

- 优先向上脱离地下
- 会尝试挖开脚下到头顶上方的可挖掘方块
- 不再出现持续在方块中快速下沉的问题
- 如果长时间无法正常出土，会回到最近记录的安全位置，避免卡死

### 5. 镐子与普通挖矿规则

当模组启用后：

- 普通手动挖矿会被禁用
- 使用镐子挖矿会被提示“镐子已经失效”
- 必须进入掘进模式才能进行地下破坏

也就是说，本模组启用后，传统挖矿会被“穿山甲掘进系统”接管。

### 6. 地下交互限制

在潜地状态下：

- 不能正常攻击实体
- 不能正常使用大多数物品
- 不能正常与大多数方块交互

例外：

- 在掘进模式下，可以放置方块
- 放置方块会消耗体力

### 7. 耐力系统

模组使用耐力作为核心资源。

耐力会影响：

- 进入掘进模式
- 自动掘进破坏方块
- 地下放置方块
- 出土脱困时清障

默认规则：

- 基础最大耐力：`100`
- 耐力恢复速度：`0.2 / tick`
- 掘进进入消耗：`6`
- 地下放置方块消耗：`4`

说明：

- `0.2 / tick` 大约等于每秒恢复 `4` 点耐力

### 8. 耐力升级

按 `H` 可以打开耐力升级界面。

升级规则：

- 每次升级增加 `50` 点最大耐力
- 最大上限为 `1000`
- 升级消耗经验等级

默认升级成本：

- 第一次升级：`3` 级
- 后续每一档在上一档基础上额外增加 `2` 级

示例：

- `100 -> 150`：消耗 `3` 级
- `150 -> 200`：消耗 `5` 级
- `200 -> 250`：消耗 `7` 级

升级完成后：

- 最大耐力立即提升
- 当前耐力会直接补满到新的上限
- 升级结果会保存到配置数据中

### 9. 头盔等级系统

头盔决定：

- 能否进入掘进模式
- 可挖掘方块等级
- 挖掘速度
- 挖掘耐力消耗倍率

当前头盔分级如下：

- `无头盔`：不能掘进
- `基础头盔`：皮革、锁链、金、海龟头盔
- `铁头盔`
- `钻石头盔`
- `下界合金头盔`

对应参考镐等级：

- 基础头盔：等同石镐
- 铁头盔：等同铁镐
- 钻石头盔：等同钻石镐
- 下界合金头盔：等同下界合金镐

### 10. 氧气与窒息规则

当玩家处于潜地状态并且身体仍在地下介质中时：

- 空气不会自然恢复
- 会持续消耗空气值
- 空气耗尽后会受到溺水伤害

这意味着：

- 地下潜行并不是无风险的
- 长时间停留在土层/石层中会有窒息压力

### 11. 回血限制

玩家在潜地状态下：

- 自然回血会被禁用

这会进一步提高地下行动的风险。

## 可潜入介质与不可潜入方块

### 可潜入介质

默认来说，只要方块：

- 不是空气
- 不是液体
- 不属于硬禁用标签
- 不带方块实体

就可以视为潜地介质。

### 不可潜入方块

默认硬禁用方块包括：

- 基岩
- 屏障
- 黑曜石
- 哭泣的黑曜石
- 强化深板岩
- 末地传送门
- 末地折跃门
- 下界传送门
- 末地传送门框架
- 结构空位
- 命令方块系列
- 拼图方块
- 重生锚

此外：

- 含液体的方块位置不能潜
- 带方块实体的方块不能潜

## 配置文件说明

配置文件路径：

```text
config/pangolin-survival.json
```

主要配置项如下：

### `globalEnabled`

- 是否启用全局玩法
- 默认：`false`

### `maxStamina`

- 基础最大耐力
- 默认：`100`

### `digEnterCost`

- 进入掘进模式需要消耗的耐力
- 默认：`6`

### `blockBreakStaminaMultiplier`

- 方块掘进耐力消耗倍率
- 默认：`7.5`

### `blockPlaceCost`

- 潜地状态下放置方块消耗的耐力
- 默认：`4`

### `staminaRecoveryPerTick`

- 每 tick 恢复的耐力
- 默认：`0.2`

### `ghostFlySpeed`

- 幽潜状态下的移动速度参数
- 默认：`0.03`

### `digFlySpeed`

- 掘进状态下的移动速度参数
- 默认：`0.02`

### `playerMaxStamina`

- 玩家个人最大耐力存档
- 会按玩家 UUID 保存

## 示例配置

```json
{
  "globalEnabled": true,
  "maxStamina": 100,
  "digEnterCost": 6,
  "blockBreakStaminaMultiplier": 7.5,
  "blockPlaceCost": 4,
  "staminaRecoveryPerTick": 0.2,
  "ghostFlySpeed": 0.03,
  "digFlySpeed": 0.02,
  "playerMaxStamina": {}
}
```

## 常见问题

### 为什么我不能用镐子挖矿？

因为模组启用后，普通挖矿被接管了，你必须进入掘进模式。

### 为什么按 G 或 V 没反应？

通常有以下几种原因：

- 服务端没有启用全局规则
- 你没有安装客户端模组
- 你附近没有可潜入介质
- 你在水里、岩浆里或载具上
- 进入掘进时头盔等级不足或体力不足

### 为什么我在地下会掉氧气？

因为潜地状态下在地下介质中不会恢复空气，长时间停留会窒息。

### 为什么活着时丢掉新手帽不会自动补发？

这是当前设计。新手帽只在首次进入世界和死亡后缺失时补发。
